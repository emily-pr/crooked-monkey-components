02 — How to call it

Reuse it on a new page

Radius tokens

--r-pill:    999px;                    /* buttons, pills, tags */
--r-card:    clamp(18px,1.6vw,26px);   /* standard cards, form container */
--r-card-lg: clamp(22px,2.4vw,40px);   /* large feature panels */
--r-sm:      10px;                     /* small chips, inputs */
--r-flat:    6px;                      /* near-straight edge (notch-card cut side) */

/* use: border-radius: var(--r-card); */
03 — When to use it

Notes & gotchas

  • Pick by role: pills/tags → --r-pill; cards/inputs → --r-card/--r-sm; feature panels → --r-card-lg.
  • --r-flat is the near-straight corner on the notch card's cut side.
