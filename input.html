02 — How to call it

Reuse it on a new page

Emit the CSS once, then render

import cm_kit as cm
style = "<style>" + cm.root_css() + cm.input_css() + "</style>"

cm.field("Email", name="email", type="email",
         placeholder="you@company.com")

# class: .cm-field (label + input/textarea)
03 — When to use it

Notes & gotchas

  • Label sits above in uppercase Label style; white fill, --r-sm radius, ink text.
  • Focus: ink border + soft ink ring. Placeholder is ink at ~45% opacity (no grey).
