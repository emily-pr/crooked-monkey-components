02 — How to call it

Reuse it on a new page

CSS once, render with your logo, wire the JS once

import cm_kit as cm
style  = "<style>" + cm.root_css() + cm.nav_css() + "</style>"

html   = cm.nav(logo_src,                      # data URI / path to cream logo
    links=["Products"],
    services=["Custom Screen Printing", "Embroidery & DTG"],
    brands=["Patagonia", "YETI", "Lululemon"])

script = "<script>" + cm.nav_js() + "</script>"  # click-to-pin dropdowns
03 — When to use it

Notes & gotchas

  • Production is position:fixed. The kit CSS leaves positioning to you; the demo above overrides to static so it sits in the page.
  • Dropdowns open on hover (CSS) and pin on click (JS); Escape closes. Triggers carry aria-expanded / aria-haspopup.
  • Ink bar, cream logo, Poppins nav links (hover yellow), yellow CTA pill. Collapses toward a drawer on narrow screens.
